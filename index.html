Discord bot

Setup guide

  1. Go to discord.com/developers/applications
  2. Create a new application → Bot → Reset Token → copy it above
  3. Under Privileged Intents, enable Message Content Intent
  4. Invite the bot to your server with Send Messages permission
  5. In Discord, enable Developer Mode (Settings → Advanced), then right-click your channel → Copy Channel ID
